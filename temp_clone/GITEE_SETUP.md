## 📋 接下来的步骤

### 第一步：获取Gitee私人令牌

1. 登录 Gitee：https://gitee.com
2. 点击头像 → 设置
3. 左侧菜单找「私人令牌」
4. 点击「生成新令牌」
5. 勾选 **projects** 权限
6. 点击提交，复制令牌

---

### 第二步：上传网站文件到仓库

把 `portfolio_project` 文件夹里的**所有文件**上传到仓库：

1. 进入仓库：https://gitee.com/z1832698917/portfolio
2. 点击「上传文件」
3. 把所有文件拖入上传区
4. 提交

---

### 第三步：开启Gitee Pages

1. 在仓库页面，点击「服务」
2. 选择「Gitee Pages」
3. 分支选择 **master**
4. 点击「启动」
5. 等待1-2分钟

---

### 第四步：访问你的网站

部署完成后访问：
```
https://z1832698917.gitee.io/portfolio
```

---

## 📱 管理后台

网站上线后，管理后台地址：
```
https://z1832698917.gitee.io/portfolio/admin/
```

**首次使用需要：**

3. 以后上传图片会自动存到你的Gitee仓库
